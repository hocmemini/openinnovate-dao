# Governance Health Check — Interview Guide

**Duration:** 30 minutes
**Deliverable:** 1-page findings summary (email within 24 hours)
**Cost:** Free (lead generation for paid audit)

---

## Pre-Call Preparation (5 min)

Before the call, gather:
- [ ] Their governance docs (constitution, charter, bylaws) — usually on GitHub or governance forum
- [ ] Their proposal/voting platform (Snapshot, Tally, Governor contract, forum)
- [ ] Last 5-10 governance proposals (titles, outcomes, participation rates)
- [ ] Treasury address (Etherscan/block explorer)
- [ ] Any public governance incidents or controversies (search Twitter, forum)

---

## Interview Structure

### 1. Governance Framework (10 min)

**Constitutional Foundation:**
- [ ] Do you have a written constitution or governance charter? Where does it live?
- [ ] How was it ratified? Has it ever been amended?
- [ ] Are governance rules enforceable or aspirational?

**Decision-Making Process:**
- [ ] How are proposals created? Who can submit?
- [ ] What is the voting mechanism? (token-weighted, quadratic, conviction, multisig)
- [ ] What is quorum? What are approval thresholds?
- [ ] Is there a timelock or delay before execution?

**Role Clarity:**
- [ ] Who can execute approved proposals? (multisig signers, guardian, council)
- [ ] Is there a clear separation between proposal, voting, and execution?
- [ ] What happens if the execution team disagrees with a vote outcome?

### 2. Ostrom's Principles Assessment (10 min)

Score each principle 0-3 (0=absent, 1=weak, 2=adequate, 3=strong):

| # | Principle | Score | Notes |
|---|-----------|-------|-------|
| 1 | **Clearly defined boundaries** — Who is a member? What resources are governed? | | |
| 2 | **Rules match local conditions** — Governance structure fits the organization's actual needs | | |
| 3 | **Collective choice arrangements** — Those affected by rules can participate in modifying them | | |
| 4 | **Monitoring** — Behavior of members and resource conditions is observed | | |
| 5 | **Graduated sanctions** — Violations have proportional consequences | | |
| 6 | **Conflict resolution** — Low-cost, accessible mechanisms for resolving disputes | | |
| 7 | **Right to organize** — External authorities recognize the community's right to self-govern | | |
| 8 | **Nested enterprises** — Governance activities are organized in multiple layers | | |

**Ostrom Score: ___ / 24**

### 3. Risk Assessment (5 min)

**Concentration Risk:**
- [ ] What is the key-person risk? (bus factor)
- [ ] Is treasury controlled by multisig? How many signers? What threshold?
- [ ] Can any single actor block governance?

**Participation Health:**
- [ ] What is typical voter participation rate?
- [ ] Is participation declining, stable, or growing?
- [ ] Are there token delegation mechanisms?

**Upgrade/Amendment Risk:**
- [ ] Can governance rules be changed? By whom?
- [ ] Is there a timelock on changes?
- [ ] Has the governance system been upgraded? How did that go?

### 4. Close (5 min)

**Summarize findings:**
- "Based on what you've shared, your three biggest governance gaps are..."
- "Your Ostrom score is X/24, which puts you in the [weak/moderate/strong] range"
- "The areas where a full audit would add the most value are..."

**Transition to paid engagement:**
- "A full governance audit would give you a structured report with constitutional assessment, decision quality analysis, and specific recommendations — all grounded in established governance theory"
- "We run your governance documents and recent proposals through our constitutional evaluation engine and deliver a reasoning tree for each one"
- "The audit starts at $1,500 and takes about a week. Want me to send a scope document?"

## What to Collect for Full Audit

If they want to proceed, request:
1. All governance documents (constitution, charter, bylaws, operating agreement)
2. Governance forum access or export of last 20 proposals
3. Treasury address(es) and any financial governance policies
4. Org chart or role descriptions for governance participants
5. Any prior governance audits or assessments
6. Known governance incidents they want specifically assessed
